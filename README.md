# zh-mobile-notes

## 项目简介

本仓库用于归档和发布一批独立的 HTML 页面。这些页面以移动端浏览为主要场景，以笔记、记录、信息展示为主要内容，不针对任何特定网站或域名。

每个页面均为自包含的 HTML 文件，可直接在浏览器中打开查看，无需额外依赖或服务器环境。

## 目录说明

```
zh-mobile-notes/
│
├── index.html          # 仓库页面入口，提供文件索引
├── pages/              # 存放所有 HTML 页面文件
│   ├── note-001.html
│   ├── note-002.html
│   └── ...
│
└── assets/             # 页面可能用到的资源文件（如 CSS、图片等）
    ├── style.css
    └── images/
```

- `index.html`：仓库的主页，通常包含页面列表或导航链接。
- `pages/`：所有归档的 HTML 页面存放于此，按需命名。
- `assets/`：页面共享或专用的资源文件，便于统一管理。

## 页面归档说明

- 每个页面均为独立的 HTML 文档，内容以文字、列表、表格等结构化信息为主。
- 页面设计优先适配移动端屏幕，但在桌面端也可正常浏览。
- 页面不包含外部追踪、广告、跳转链接或商业推广内容。
- 归档内容可能不定期更新、修正或补充，不保证长期不变。
